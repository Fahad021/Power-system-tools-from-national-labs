# Power-system-tools-from-national-labs

A living catalog of power-systems tools and projects from national labs in the **United States, Canada, and Mexico**, covering major subfields (planning, operations, markets, DERs, resilience, cybersecurity, electrification, and data).

> **Scope note:** This repository is designed to be comprehensive over time. It starts with widely known/publicly documented tools and projects and should be expanded continuously.

## How this repository is organized

- By **country**
- By **lab/institution**
- By **subfield** (planning, operations, DERs, markets, resilience, etc.)
